# Sovela

> Regenerative capital for economic actors whose capability 
> is real but invisible to formal finance.

Sovela connects economic actors with aligned capital providers 
through trust-based covenants — profit-sharing agreements 
that replace extractive lending with shared growth. When a 
business succeeds, everyone who believed in it succeeds with it.

Built on the [Zivana Protocol](https://github.com/zivana-labs) — 
open trust infrastructure for the African informal economy.

---

## The Problem Sovela Solves

Nigeria alone has approximately 40 million micro, small, and 
medium enterprises contributing 58% of national GDP. The majority 
cannot access formal capital — not because they lack capability, 
but because their capability is invisible to funders outside their 
immediate community.

Sovela makes that capability visible, verifiable, and financeable. 
A trader in Lagos with five years of consistent sales history, 
hundreds of loyal customers, and a reputation her community trusts 
can now demonstrate all of that — to a capital provider anywhere 
in the world — through a verifiable, privacy-preserving trust 
profile she owns and controls.

---

## How It Works

**Performance Trust** — Economic actors with existing marketplace 
presence build credibility through verifiable transaction data, 
customer reviews, and fulfilment records. Their trust profile is 
on-chain, holder-owned, and portable across every application 
built on Zivana Protocol.

**Human Trust** — Actors without transaction history access 
capital through Agent sponsorship. Established community members 
vouch for individuals they believe in, earning a share of the 
value generated by the businesses they helped launch.

**The Covenant** — When funding is agreed, a smart contract 
covenant formalises the terms. When the business generates 
profits, value distributes automatically across all participants 
— capital providers receive proportional returns, the business 
retains the majority for reinvestment, the Agent earns a 
commission, and the protocol treasury sustains the infrastructure.

No fixed interest. No extraction. Aligned growth.

## Built on Zivana Protocol

Sovela is an application, not the infrastructure. It uses the 
[Zivana Protocol SDK](https://github.com/zivana-labs/zivana-sdk) 
to interact with four protocol primitives:

| Primitive | What Sovela uses it for |
|---|---|
| Identity | Participant credentials with Nigerian jurisdiction compliance |
| Trust Score | On-chain reputation engine across four dimensions |
| Covenant | Profit-sharing agreements between economic actors and capital providers |
| Distribution | Automated value split triggered by oracle-attested revenue events |
| Intelligence | Market benchmarking, covenant viability scoring, anomaly detection |

## Application Stack

**Frontend** — Three Next.js 14 applications:
- Capital Provider Dashboard — portfolio tracking, opportunity discovery, multi-currency wallet
- Economic Actor Portal — funding requests, profit reporting, trust profile management
- Admin Platform — verification workflows, operational oversight, analytics

**Backend** — Node.js / Express / Prisma / PostgreSQL

**Blockchain** — Cardano (Aiken contracts), Midnight Network 
(Compact contracts), Hyperledger Identus (identity layer)

**Infrastructure** — Vercel (frontend), Railway / AWS (backend), 
Blockfrost (Cardano API)

## Getting Started

```bash
git clone https://github.com/sovela-hq/sovela.git
cd sovela

# Backend
cd backend && npm install
cp .env.example .env
npx prisma migrate dev
npm run dev

# Frontend — capital provider dashboard
cd capital-dashboard && npm install
cp .env.local.example .env.local
npm run dev
```
